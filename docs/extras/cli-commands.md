# CLI commands

```
$ ./privado 
Privado is a CLI tool that scans & monitors your repositories to build privacy, transparency reports & finds privacy issues. 
Find more at: https://github.com/Privado-Inc/privado

Usage:
  privado [command]

Available Commands:
  auth        Request license for Privado
  bootstrap   Authenticates Privado using the requested license and generates required configurations
  completion  Generate the autocompletion script for the specified shell
  help        Help about any command
  load        Load a scanned codebase to continue generating compliance reports
  scan        Scan a codebase or repository to identify privacy issues and generate compliance reports

Flags:
  -h, --help             help for privado
  -l, --license string   The license file to be used. Overrides the default bootstrapped license (default "/Users/ojaswa/.privado/license.json")

Use "privado [command] --help" for more information about a command.
```
